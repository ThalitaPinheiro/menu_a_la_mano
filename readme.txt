== Description ==

Plugin para cadastro de cardápio e carta de vinhos para restaurante.
É possível criar categorias e subcategorias para o cardápio e para a carta de vinhos.
As informações para o cardápio são: Nome, descrição, preço, data de entrada e saída do menu.
As informações para o vinho são: Nome, descrição, preço e ano.

== Installation ==

1. Upload `menu_a_la_mano.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
